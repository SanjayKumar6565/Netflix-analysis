# 🎬 Netflix Data Analysis Project

## 🔹 Project Overview
This project analyzes Netflix movies and TV shows dataset to uncover trends, content distribution, and growth patterns.

---

## 📸 Project Preview
![Dashboard](netflix_dashboard.png)

---

## 🔍 Key Insights
- Distribution of Movies vs TV Shows  
- Top countries producing Netflix content  
- Year-wise content growth  
- Most common genres on Netflix  

---

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- Excel

---

## 📂 Workflow
1. Data cleaning using Pandas  
2. Exploratory Data Analysis (EDA)  
3. Visualization using Matplotlib  

---

## 📁 Files Included
- `netflix_analysis.py` → Python analysis  
- `netflix_titles.csv` → dataset  

---

## 🚀 Conclusion
This project demonstrates how data analysis can uncover trends in streaming platforms and user preferences.
